A Logo simulation of the heat bugs model. See http://ccl.northwestern.edu/netlogo/models/Heatbugs for more information.

The main class of this simulation model is defined in the 'HeatBugsSimulationMain.java' file, located in the 'examples/heatbugs/src/java/fr/univ_artois/lgi2a/similar2logo/examples/heatbugs' directory of the distribution.

To run this simulation, use the following command from the root directory of the distribution :
	java -cp "lib/*" fr.univ_artois.lgi2a.similar2logo.examples.heatbugs.HeatBugsSimulationMain
