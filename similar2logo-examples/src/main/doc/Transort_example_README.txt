A Logo simulation of tranport on the town of Valenciennes. Peoples, cars, tramways and trains cross the town for simulating the daily life.

The main class of this simulation model is defined in the 'TransportSimulationMain.java' file, located in the 'similar2logo-examples\src\main\java\fr\lgi2a\similar2logo\examples\transport' directory of the discribution.

To run this simulation, use the following command from the root directory of the distribution :
	jave -cp "lib/*" fr.lgi2a.similar2logo.examples.transport.TransportSimulationMain
