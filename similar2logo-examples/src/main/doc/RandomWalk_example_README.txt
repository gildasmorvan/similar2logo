A Logo simulation with a turtle randomly walking in the environment.

The main class of this simulation model is defined in the 'RandomWalkSimulationMain.java' file, located in the 'examples/randomwalk/src/java/fr/univ_artois/lgi2a/similar2logo/examples/randomwalk' directory of the distribution.

To run this simulation, use the following command from the root directory of the distribution :
	java -cp "lib/*" fr.univ_artois.lgi2a.similar2logo.examples.randomwalk.RandomWalkSimulationMain
